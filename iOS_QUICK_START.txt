┌─ FULL TESTING (10 MINUTES) ──────────────────────────────────────────────┐
│                                                                            │
│  1. open /Applications/Xcode.app                                           │
│  2. Create New Project:                                                   │
│     File → New → Project → iOS → App                                      │
│     Product Name: MindfulLiving                                           │
│     Language: Swift, Interface: SwiftUI                                   │
│                                                                            │
│  3. Copy SwiftUI files:                                                   │
│     cp -r ios/MindfulSwiftUI/* YourXcodeProject/YourXcodeProject/         │
│                                                                            │
│  4. Run on simulator:                                                     │
│     Product → Destination → iPhone 16 Pro Max                             │
│     Product → Run (⌘R)                                                    │
│                                                                            │
│  5. Test all screens:                                                     │
│     Dashboard → Explore → Detail → Journal → Practices → Profile          │
│                                                                            │

│                                                                            │
